# LinkUp 🔗
### "Any skill. Any task. Right around you."

LinkUp is a hyperlocal gig economy platform where anyone can post a job and anyone nearby can apply. Think of it as a community-based task marketplace — no formal qualifications needed, just skills and willingness to help.

## 🌟 Features
- 📍 Location-based job feed with radius filter
- 💼 Post any job — plumbing, tutoring, design, delivery, anything
- 🔴 Urgent job tags for immediate help
- 💬 In-app chat between poster and applicant
- ⭐ Trust score system built from reviews
- 🌙 Dark/Light mode toggle
- 📱 Fully responsive — mobile + desktop
- 🔐 Secure authentication via Supabase
- 💰 Budget in ₹ Rupees

## 🚀 Tech Stack
- React + Next.js
- Tailwind CSS
- Supabase (Database + Auth)
- Vercel (Deployment)
- check out on https://link-up-gigs.vercel.app/
## 🛠️ Getting Started
1. Clone the repo
\`\`\`bash
git clone https://github.com/Mohamin007/LinkUP.git
\`\`\`
2. Install dependencies
\`\`\`bash
npm install
\`\`\`
3. Create `.env.local` file
\`\`\`
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
\`\`\`
4. Run locally
\`\`\`bash
npm run dev
\`\`\`

## 💡 Use Cases
- Need a plumber right now? Post it.
- Want assignment help? Post it.
- Looking for a local salesman? Post it.
- Any skill. Any task. Any time.

## 🎯 How It Works
1. **Post a Job** — Describe task, set budget, set radius
2. **Get Applicants** — Nearby people apply instantly
3. **Chat & Hire** — Chat, agree, get it done
4. **Rate & Review** — Build community trust
